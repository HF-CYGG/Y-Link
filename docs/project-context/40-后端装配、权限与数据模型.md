# Y-Link 后端装配、权限与数据模型

## 最后核对来源

- `backend/src/index.ts`
- `backend/src/app.ts`
- `backend/src/config/data-source.ts`
- `backend/src/middleware/auth.middleware.ts`
- `backend/src/middleware/client-auth.middleware.ts`
- `backend/src/routes/*`
- `backend/src/services/*`
- `backend/src/entities/*`

## 适用范围

- 适用于后端启动、鉴权、安全、系统配置、通知、迁移与实体关系的首轮定位。
- 本页是总览与路由页，不承担所有服务细节。

## 模块边界摘要

- 后端可拆为四个重域：
  - 启动装配、路由装配与健康检查
  - 权限、Cookie、CSRF、审计与上传安全
  - 系统配置、通知中心与数据库迁移
  - 核心实体与关键数据关系
