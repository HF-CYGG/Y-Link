## 适用范围

- 适用于管理端工作台、出库、报表、产品中心、入库、系统管理、客服工作台等模块的入口定位。
- 本页只做域地图与阅读分发，不替代深度业务文档。

## 模块边界摘要

- 管理端主业务可分为四个重域：
  - 工作台、报表与出库链路
  - 产品、SKU、标签与 O2O 商品管理
  - 入库、供应方与库存核验
  - 系统管理、用户中心与客服工作台
- 它们共享同一个管理端壳层，但状态流转、权限点和服务真源完全不同。
