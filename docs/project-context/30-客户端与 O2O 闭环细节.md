# Y-Link 客户端与 O2O 闭环细节

## 最后核对来源

- `src/router/routes.ts`
- `src/views/client/*`
- `src/views/o2o/*`
- `src/store/modules/client-auth.ts`
- `src/store/modules/client-cart.ts`
- `backend/src/routes/o2o.routes.ts`
- `backend/src/routes/client-feedback.routes.ts`

## 适用范围

- 适用于客户端登录、商城、购物车、结算、预订单、退货、反馈会话和管理端 O2O 协同的入口定位。
- 本页是总览与跳转页，不承担超大闭环的全部实现细节。

## 模块边界摘要

- 客户端/O2O 可拆为三个重域：
  - 客户端鉴权、商城、购物车与结算
  - O2O 预订单、核销、退货与正式出库联动
  - 反馈会话与实时客服闭环
- 客户端与管理端共用部分价格和库存口径，但会话体系、权限体系和页面入口不同。
