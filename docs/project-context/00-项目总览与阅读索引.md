## 适用范围

- 适用于新会话第一次进入 `F:\Y-Link` 时的硬入口。
- 适用于先判断需求应落到前端平台层、管理端业务域、客户端/O2O、后端基础设施还是部署验证链路。
- 不直接替代代码，也不替代子文档；它负责把阅读顺序和跳转关系固定下来。

## 模块边界摘要

- 前端主仓位于 `src/`，采用 Vue 3 + TypeScript + Pinia + Vue Router + Element Plus。
- 后端主仓位于 `backend/src/`，采用 Express + TypeORM，支持 SQLite 与 MySQL 双模式。
- 管理端、客户端、供应方入口共用同一仓库，但会话、权限、库存和状态机并不完全相同。
- 高复杂度代码主要集中在前端平台层、管理端订单/报表/产品/入库、客户端 O2O/反馈，以及后端系统配置/通知/迁移。

## 新会话默认读取规则

1. 先读本文件。
2. 再读对应总览页：`10`、`20`、`30`、`40`、`50`。
3. 最后根据任务关键词直达一个或多个子文档。
4. 若任务跨多个主题，可以组合阅读多个总览页和子文档。
5. 若文档与代码冲突，以代码为准，并优先回写文档。
6. 禁止只凭记忆回答 Y-Link 仓库问题。
