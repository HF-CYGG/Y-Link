## 阶段总结

- `v1.2.x`：从应用元数据、客户端基础体验起步，快速搭建客服反馈闭环，并补齐供货方改单与扫码入库辅助能力。
- `v1.3.x`：重点转向认证安全、上传路径兼容、性能治理、通知中心与客户端会话管理，逐步形成可上线的生产化能力。
- `v1.4.x`：进入业务扩展高峰期，完成客户端账号体系、教职工目录、系统配置、报表中心、O2O 折扣与 SKU 规格链路的大量迭代。
- `v1.5.0`：开始强调交付体验与视觉品质，补强 onebox 上传验证，并升级商城视觉与 README 预览展示。
